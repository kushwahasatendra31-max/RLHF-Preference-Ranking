# Preference Ranking Example

## Prompt
Explain machine learning.

## Response A
Machine learning is a branch of AI that enables systems to learn from data.

## Response B
Machine learning makes computers smart.

## Preferred Response
Response A

## Reason
More accurate, complete, and informative.
